# Hospital_DA_sql
# Hospital Revenue & Patient Analytics

## Project Overview
This project focuses on analyzing hospital patient and revenue data to understand treatment patterns, revenue drivers, and patient demographics. Using SQL and Python, the analysis aims to support data-driven decision-making for hospital operations and resource utilization.

## Dataset
The dataset includes structured hospital records such as:
- Patient details
- Treatment and diagnosis information
- Admission and discharge dates
- Billing and payment status
- Revenue generated per treatment

Data was cleaned and normalized before analysis to ensure accuracy.

## Tools & Technologies
- SQL (MySQL)
- Python (Pandas, NumPy)
- Exploratory Data Analysis (EDA)
- Data Cleaning & Aggregation

## Key Business Questions Answered
- Which treatments generate the highest revenue?
- What is the distribution of paid vs unpaid patients?
- How does patient stay length vary across treatments?
- Which patient demographics contribute most to hospital revenue?
- What factors influence hospital resource utilization?
